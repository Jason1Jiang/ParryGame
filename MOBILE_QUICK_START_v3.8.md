# 手机端快速开始 v3.8

## 🎮 如何在手机上玩

### 1. 打开游戏
在手机浏览器中打开游戏网址

### 2. 开始游戏
点击"开始游戏"按钮

### 3. 操作方式
- **格挡** - 点击屏幕任意位置
- **移动** - 无法主动移动，只能通过反击瞬移
- **反击** - 格挡成功后自动瞬移反击

### 4. 游戏目标
依靠精准的格挡时机和反击击败所有敌人，存活尽可能长的时间

## 💡 游戏提示

### 移动端特点
1. **无法移动** - 你无法主动移动，只能依靠格挡反击
2. **屏内刷怪** - 所有敌人在屏幕内刷新，不会从屏幕外攻击
3. **更多反应时间** - 敌人距离更远，警告时间更长
4. **触摸反馈** - 点击时会显示涟漪效果

### 操作技巧
1. **时机很重要** - 在敌人攻击瞬间格挡可触发完美格挡
2. **完美格挡** - 可以触发多次反击，一次击败多个敌人
3. **能量管理** - 格挡消耗能量，注意能量条
4. **反击瞬移** - 利用反击改变位置，躲避危险

## 🔧 常见问题

### Q: 为什么无法移动？
A: 移动端设计为纯格挡模式，无法主动移动，只能通过反击瞬移改变位置。

### Q: 如何移动到其他位置？
A: 格挡敌人攻击后，会自动瞬移到敌人位置并击杀敌人。

### Q: 游戏太难了怎么办？
A: 移动端已经降低了难度（敌人距离更远，警告时间更长），多练习格挡时机。

### Q: 支持横屏吗？
A: 支持！旋转手机到横屏，游戏会自动调整布局。

### Q: 为什么点击没反应？
A: 确保点击的是游戏画布区域，不要点击 UI 元素。
