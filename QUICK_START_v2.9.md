# 🎮 v2.9 快速启动指南

## 新功能速览

### ⚙️ 调试界面
按 **F3** 查看实时游戏数据！

### 🎯 难度选择
三种难度任你选择：
- ⚡⚡⚡ **硬核**: 极限挑战
- ⚡⚡ **平衡**: 推荐选择
- 🌙 **休闲**: 轻松体验

---

## 快速开始

### 1. 启动游戏
```bash
# Windows
双击 start.bat

# 或使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```

### 2. 选择难度
1. 点击 **"开始游戏"**
2. 选择合适的难度
3. 开始战斗！

### 3. 查看调试信息
- 按 **F3** 键
- 或点击右上角 **"调试 [F3]"** 按钮

## 调试数据说明

### 🔴 红色 = 极快/危险
- 刷怪间隔 < 600ms
- 连击数 ≥ 10x
- 敌人过多

### 🟠 橙色 = 快速/警告
- 刷怪间隔 600-1200ms
- 连击数 ≥ 5x
- 敌人不足

### 🟢 绿色 = 正常/良好
- 刷怪间隔 > 1200ms
- 敌人数量合适

## 进阶技巧

3. 连续斩击多个敌人

## 常见问题

**Q: 如何切换难度？**
A: 游戏结束后重新开始，选择新难度。

**Q: 调试界面影响性能吗？**
A: 几乎不影响（+0.5% CPU）。

**Q: 如何关闭调试界面？**
A: 再次按 F3 键。
