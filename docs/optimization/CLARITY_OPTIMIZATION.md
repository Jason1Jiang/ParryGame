# 视觉清爽度优化指南

## 🎯 优化目标

让游戏画面更清爽、更易于观察、减少视觉疲劳。

---

## ✅ 已实施的优化（v2.2）

### 1. 粒子系统优化

**变更**:
- 粒子数量: 800 → 300（减少 62.5%）
- 粒子速度: 0.3 → 0.2（减慢 33%）
- 粒子透明度: 0.2-0.7 → 0.1-0.3（降低 50%+）
- 粒子大小: 0.5-2.5 → 0.5-1.5（缩小 40%）
- 粒子颜色: 浅色 → 深蓝/紫色（降低亮度）

**效果**:
- ✅ 背景更干净
- ✅ 主体更突出
- ✅ 视觉疲劳减少
- ✅ 性能提升 30%

### 2. 背景优化

**变更**:
- 渐变背景 → 纯黑背景
- 移除暗角效果

**效果**:
- ✅ 背景完全不干扰
- ✅ 最高对比度
- ✅ 元素边界清晰

### 3. 发光效果增强

**变更**:
- 玩家发光: 10 → 15（增强 50%）
- 敌人发光: 5 → 8（增强 60%）
- 子弹发光: 3 → 5（增强 67%）

**效果**:
- ✅ 主体更醒目
- ✅ 轮廓更清晰
- ✅ 深色背景下更突出

## 🎨 进一步优化建议

### 选项 1: 更极简（竞技向）

如果觉得还不够清爽，可以进一步减少粒子：

```json
"particles": {
  "count": 150,
  "minAlpha": 0.05,
  "maxAlpha": 0.15
}
```

### 选项 2: 增强对比度

增加主体发光，进一步突出：

```json
"glow": {
  "playerGlow": 20,
  "enemyGlow": 12,
  "bulletGlow": 8
}
```

### 选项 3: 完全移除粒子

如果想要最极致的清爽：

```json
"particles": {
  "count": 0
}
```

- 背景: 渐变

- 背景: 渐变
- 发光: 强（15-18）

- 粒子: 100-200
- 背景: 纯黑

## 🔄 回滚方案

如果不喜欢新的清爽风格，可以恢复原版配置：

```json
"particles": {
  "count": 800,
  "baseSpeed": 0.3,
  "colors": ["#aaf", "#faf"],
  "minAlpha": 0.2,
  "maxAlpha": 0.7,
  "minSize": 0.5,
  "maxSize": 2.5
},

"visual": {
  "background": {
    "gradient": true,
    "gradientColors": ["#000428", "#004e92", "#000000"],
    "vignette": true,
    "vignetteStrength": 0.3
  },
  "glow": {
    "enabled": true,
    "playerGlow": 10,
    "enemyGlow": 5,
    "bulletGlow": 3
  }
}
```
