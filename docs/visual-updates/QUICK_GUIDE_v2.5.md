# 多重反击系统 - 快速指南

## 什么是多重反击？

在按下格挡键的**极短时间内**（150ms）成功格挡敌人攻击，会触发**多重反击**，连续瞬移斩击最多3个敌人。

## 如何触发？

```
1. 观察敌人即将攻击
2. 在攻击命中前的瞬间按下空格键
3. 成功格挡 → 触发多重反击！
```

### 最佳时机

- **近战敌人**: 警告动画结束的瞬间
- **远程敌人**: 子弹即将命中的瞬间

## 视觉效果

- 🟣 **紫色闪光** - 超强闪光效果
- 🟣 **紫色刀光** - 独特的紫色瞬移轨迹
- ⏱️ **超慢时间** - 0.05倍速慢动作（持续500ms）
- 💥 **超强震动** - 强度20（是普通格挡的2.5倍）
- 📝 **飘字提示** - "MULTI COUNTER x3!"

## 奖励

- ⚡ **能量恢复**: +20基础 + 每个敌人+18 = 最多74能量
- 🎯 **清场能力**: 一次性击杀3个敌人
- 🔥 **连击维持**: 快速累积连击数

## 技巧

### 新手
1. 先练习近战敌人（更容易预判）
2. 观察警告动画，在结束瞬间按键
3. 保持足够能量维持格挡

### 进阶
1. 被多个敌人围攻时使用
2. 利用慢动作观察战场
3. 配合连击系统获得更强效果

### 高手
1. 连续触发多重反击
2. 精准控制时间窗口
3. 能量管理与战术配合

## 配置调整

在 `config.json` 中找到 `multiCounter` 配置：

```json
{
  "timeWindow": 150,      // 时间窗口（毫秒）
  "maxTargets": 3,        // 最大目标数
  "bonusEnergy": 20       // 能量奖励
}
```

### 降低难度
- `timeWindow`: 150 → 200（更长的时间窗口）
- `maxTargets`: 3 → 4（更多目标）

### 提高难度
- `timeWindow`: 150 → 100（更短的时间窗口）
- `maxTargets`: 3 → 2（更少目标）

## 与其他系统的区别

| 系统 | 触发条件 | 视觉效果 | 反击次数 | 能量奖励 |
|------|---------|---------|---------|---------|
| 普通格挡 | 长按空格 | 青色护盾 | 1次 | 击杀+18 |
| 完美格挡 | 连续格挡 | 金色闪光 | 1次 | +10 + 击杀+18 |
| **多重反击** | **极短时间格挡** | **紫色闪光** | **3次** | **+20 + 击杀×3** |

## 常见问题

**Q: 为什么没有触发？**
A: 可能按键太早或太晚，时间窗口只有150ms

**Q: 如何练习时机？**
A: 先对付单个近战敌人，观察警告动画

**Q: 能量不够怎么办？**
A: 多重反击会恢复大量能量，可以连续使用

**Q: 可以调整难度吗？**
A: 可以，修改 config.json 中的 timeWindow 参数
