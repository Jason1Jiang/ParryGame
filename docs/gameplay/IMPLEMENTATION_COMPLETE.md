## 🎨 核心特性

### 粒子系统
- **数量**: 100个
- **颜色**: 白色(30%) + 玩家色(50%) + 青色(20%)
- **物理**: 重力 + 空气阻力
- **寿命**: 0.5-0.8秒

### 视觉效果
- **震动**: 强度30（超强）
- **慢动作**: 0.05x速度
- **冲击波**: 半径200px
- **闪光**: 1.5倍强度

### 动画流程
```
0秒 → 触发
0.4秒 → 玩家消失
1秒 → 只有粒子
2秒 → 淡入遮罩
2.2秒 → 显示结算
```

## ⚙️ 配置调整

### 位置
`config.json` → `visual` → `playerDeath`

### 快速调整
```json
// 快速死亡（1.2秒）
"animationDuration": 1200,
"fadeOutDelay": 1200

// 更多粒子（150个）
"particleCount": 150

// 更强震动
"screenShakeIntensity": 40

// 禁用动画
"enabled": false
```
