# 调试界面使用指南

## 快速开始

### 打开调试界面
- 按 **F3** 键
- 或点击右上角 **"调试 [F3]"** 按钮

### 关闭调试界面
- 再次按 **F3** 键
- 或再次点击 **"调试 [F3]"** 按钮

---

## 调试数据说明

### 📊 刷怪间隔
```
显示: 750ms
含义: 当前每隔 750 毫秒生成一个敌人
颜色:
  🔴 红色 (< 600ms)   - 极快，高强度战斗
  🟠 橙色 (600-1200ms) - 快速，正常节奏
  🟢 绿色 (> 1200ms)   - 较慢，轻松节奏
```

**如何影响**:
- 击杀速度越快，刷怪间隔越短
- 连击数越高，刷怪间隔越短
- 敌人不足时，刷怪间隔变短

---

### ⏱️ 平均击杀
```
显示: 1000ms
含义: 最近 5 次击杀的平均间隔
用途: 衡量你的击杀速度
```

**参考标准**:
- < 800ms：高手水平 ⚡
- 800-1500ms：正常水平 ✅
- > 1500ms：新手水平 🌙

---

### 🔥 当前连击
```
显示: 5x
含义: 当前连击数
颜色:
  🔴 红色 (≥ 10x) - 超高连击
  🟠 橙色 (≥ 5x)  - 高连击
  🔵 青色 (< 5x)  - 正常
```

**连击效果**:
- 3连击：刀光强化
- 5连击：彩虹刀光
- 10连击：刷怪速度 +67%
- 15连击：刷怪速度 +100%

---

### ⚡ 连击倍率
```
显示: 0.75x
含义: 连击对刷怪速度的影响
计算: 刷怪间隔 × 连击倍率
```

**倍率对照表**:
| 连击数 | 倍率 | 刷怪速度变化 |
|--------|------|-------------|
| 0-2    | 1.0x | 无变化      |
| 3-4    | 0.9x | +11%        |
| 5-9    | 0.75x| +33%        |
| 10-14  | 0.6x | +67%        |
| 15+    | 0.5x | +100%       |

---

### 👾 场上敌人
```
显示: 6
含义: 当前场上的敌人数量
颜色:
  🔴 红色 (> 目标+3) - 敌人过多
  🟠 橙色 (< 目标-2) - 敌人不足
  🟢 绿色 (正常范围) - 数量合适
```

**数量影响**:
- 敌人过多：刷怪减速 30%
- 敌人不足：刷怪加速 30%
- 数量合适：正常刷怪

---

### 🎯 目标数量
```
显示: 5
含义: 系统期望的敌人数量
计算: 基础目标 + 游戏时间 × 增长速度
```

**目标增长**:
- 硬核模式：5 → 15（每 10 秒 +1）
- 平衡模式：4 → 12（每 10 秒 +1）
- 休闲模式：3 → 8（每 10 秒 +1）

---

### 📈 数量倍率
```
显示: 1.0x
含义: 敌人数量对刷怪速度的影响
计算: 刷怪间隔 × 数量倍率
```

**倍率规则**:
- 敌人 < 目标-2：0.7x（加速 30%）
- 敌人 > 目标+3：1.3x（减速 30%）
- 其他情况：1.0x（正常）

---

### 🎮 难度模式
```
显示: 平衡
含义: 当前选择的难度
选项: 硬核 / 平衡 / 休闲
```

## 实战案例

### 案例 1：高手玩家
```
刷怪间隔: 400ms  🔴 (极快)
平均击杀: 800ms
当前连击: 12x    🔴 (超高)
连击倍率: 0.6x
场上敌人: 8      🟢 (合适)
目标数量: 7
数量倍率: 1.0x
难度模式: 硬核
```

**分析**:
- ✅ 击杀速度快（800ms）
- ✅ 保持高连击（12x）
- ✅ 刷怪速度达到最小值（400ms）
- ✅ 敌人数量控制良好
- 💡 **建议**: 保持当前节奏，继续挑战！

---

### 案例 2：新手玩家
```

连击倍率: 1.0x

数量倍率: 1.0x
```

**分析**:

## 优化建议

### 提升击杀速度
1. 保持连击（不要断连）
2. 使用完美格挡（金色特效）
3. 触发多重反击（紫色刀光）
4. 预判敌人位置

### 控制敌人数量
1. 不要让敌人堆积
2. 优先击杀近战敌人
3. 利用多重反击清场
4. 保持移动，避免被围

### 提高连击数
1. 快速击杀敌人
2. 不要被击中
3. 保持格挡成功率
4. 利用完美格挡

## 常见问题

### Q: 为什么刷怪间隔一直在变？
A: 这是动态刷怪系统的特性，会根据你的击杀速度、连击数和敌人数量实时调整。

### Q: 如何让刷怪速度变快？
A: 
1. 提高击杀速度
2. 保持高连击
3. 清理多余敌人

### Q: 如何让刷怪速度变慢？
A: 
1. 降低击杀速度
2. 断掉连击
3. 让敌人堆积（不推荐）

### Q: 调试界面会影响性能吗？
A: 几乎不会，CPU 增加约 0.5%，可以忽略。

### Q: 可以自定义调试界面吗？
A: 可以，修改 `index.html` 中的 `#debugPanel` 样式和 `game.js` 中的 `updateDebugInfo()` 函数。
